Angular

SPA - Single Page Application

Diretivas

Diretivas de Atributo:

ngStyle

Aplica um estilo diretamente na tag

Exemplo:

(h2 [ngStyle]= "{background: count < 5 ? 'red' : 'green'}")Contador: {{ count }} (/h2)

ngClass

Aplica uma classe diretamente na tag

Exemplo:

(h2 [ngClass]= "{custom: count < 5 ? 'red' : 'green'}")Contador: {{ count }} (/h2)

Diretivas Estruturais:

ngIf

Aplica um if diretamente na tag

Exemplo:

(h2 *ngIf= "count < 7") O contador passou de 7! (/h2) Só surge na tela se a variavel count for maior que 7

Exemplo com else:

(h2 *ngIf= "count < 7 ; else menor" ) O contador passou de 7! (/h2) (ng-Template #menor)O contador é menor que 7!(/ng-Template)

ngSwitch

Aplica um switch case diretamente na tag

Exemplo:

(ul [ngSwitch] = "count") (li *ngSwitchCase= "2")É igual a 2(/li) (li *ngSwitchCase= "4")É igual a 4(/li) (li *ngSwitchCase= "6")É igual a 6(/li) (li *ngSwitchCase= "8")É igual a 8(/li) (li *ngSwitchCase= "10")É igual a 10(/li) (li *ngSwitchDefault)Padrão(/li) (/ul)

ngFor

Aplica um for diretamente na tag

Exemplo:

(ul) (li *ngFor= "let pessoa of pessoas") O nome da pessoa é {{ pessoa.nome }} (/li) (/ul)

Dentro da classe deste componente precisa ter esse vetor:

pessoas = [ { nome: andy, idade: 38 }, { nome: Jô, idade: 37 }, etc... ]
