JAVA

Linguagem no paradgma de orientação a objetos.

Linguagem interpretada

JDK - Java Development Kit

JVM - Java Virtual Machine

JRE - Java Runtime Enviroment

Variáveis em Java

  • Visibilidade (Public, protected e private)
  • Modificador (static e final)
  • Tipo de Dado
  • Nome
  • Valor Inicial

Modificador

Final - Valor imutável, definido como uma constante.

Tipo de Dado

Estática (forte) - Tipo definido desde a criação da variável

Dinânima (fraco) - Tipo é inferido no decorrer do programa

Primitivo - Tipo de Variável já existente dentro da linguagem

Composto - Composto de mais de ma variável primitiva

Textual - formada por texto

Numeral - dado numérico

Lógico - verdadeiro ou falso

Objeto - Usado em orientação a objeto

Métodos

Porção de código disponibilizada por uma classe.
